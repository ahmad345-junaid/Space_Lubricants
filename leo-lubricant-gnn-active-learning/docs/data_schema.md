# Data Schema

Required columns for supervised training:

- `molecule_id`
- `smiles`
- `temperature_c`
- `vacuum_pa`
- `radiation_gy`
- `oxygen_ppm`
- `load_n`
- `sliding_speed_mps`
- `vapor_pressure`
- `viscosity`
- `friction_coefficient`
- `wear_rate`
- `mass_loss`
- `viscosity_change`
- `degradation_index`

Candidate acquisition inputs require the identifier, SMILES, and environment columns.
