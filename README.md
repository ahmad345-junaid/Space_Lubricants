# Space Lubricants

This repository contains the project files for **Space-Environment-Conditioned Degradation-Aware Graph Neural Learning for Tribological Discovery of Low-Volatility Space Lubricants**.

The work focuses on using graph neural networks, uncertainty estimation, and active learning to identify low-volatility lubricant candidates suitable for low-Earth-orbit space environments. The framework considers key degradation factors such as vacuum exposure, atomic oxygen, UV-equivalent irradiation, thermal cycling, friction, wear, viscosity stability, and mass loss.

## Repository Status

This repository currently contains preliminary project materials related to the manuscript.
